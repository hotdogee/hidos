This analysis can present the process through which new blood vessels form from pre-existing vessels by quantified information from the image, including mesh count, junction count, extremity count, branch length, segment length, network length, and mesh area size. Cell C1

Cell counting analysis provide the precise number of cells from florescent-dyed cell image. In the past, scientists cost tremendously long time to count the number in the image manually, but, now, with Cell C1 tool, they can shrink the duration of counting and generate the data with labeled image by simple operation on the monitor, improving the efficiency of experiments and analysis.
